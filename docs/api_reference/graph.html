find_friend_cliques() → List[List[Any]]¶

Find all cliques of friends.

Danger

Note that this algorithm is exponential in the number of nodes and thus, it will not scale to big graphs. To workaround this, other approaches might be used, such as greedy or heuristic approaches. Probably for analysis purposes we are more interested in maximal cliques rather than finding all cliques.

The graph is projected into an unidirected graph and only the relationship FRIENDS_WITH is considered.

Returns

All cliques in the graph.

Return type

List[List[Any]]
